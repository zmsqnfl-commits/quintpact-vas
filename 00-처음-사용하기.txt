VAS 2.6.0 처음 사용하기
=======================

1. 받은 ZIP 파일을 새 폴더에 모두 압축 해제하세요.
   ZIP 안에서 바로 실행하면 저장이나 가져오기가 제대로 되지 않을 수 있습니다.

2. `Run-VAS-System.bat`를 더블클릭하세요.
   잠시 뒤 브라우저에서 VAS 허브가 자동으로 열립니다.

3. 허브에서 원하는 작업을 선택하세요.
   - 새 프로그램: 새 프로젝트 시작 → 신청서 작성 → 디자인 설정 → 작업공간 만들기
   - 기존 프로그램: 기존 프로젝트 가져오기 → 원래 폴더 선택 → 분석 확인 → 가져오기 승인

4. 기존 프로그램을 가져올 때
   - Python 3.10 이상이 필요합니다.
   - 분석 중에는 원본 파일을 실행하거나 수정하지 않습니다.
   - 승인 후 검증된 사본을 `workspace\projects\`에 만듭니다.
   - 원본 폴더는 자동으로 삭제하지 않습니다.
   - RAG 연결을 선택하면 허용된 문서와 코드만 이 컴퓨터 안에서 색인합니다.

5. 다시 사용할 때
   같은 폴더의 `Run-VAS-System.bat`를 다시 실행하면 최근 프로젝트에서 이어갈 수 있습니다.

6. 종료
   브라우저를 닫아도 저장된 프로젝트는 유지됩니다. 사용하지 않으면 로컬 서버는 자동 종료됩니다.

문제가 생기면
- ZIP을 완전히 풀었는지 확인하세요.
- 폴더를 쓰기 가능한 위치(문서 또는 바탕 화면 등)로 옮겨 다시 실행하세요.
- BAT 창에 표시되는 안내를 확인하세요.
- 자세한 내용은 `README.md`와 `docs\import-existing-project.md`를 확인하세요.

별도 신청서 폴더 `VAS-Client-Form-*`를 받은 경우에는 `index.html`을 더블클릭하세요.
