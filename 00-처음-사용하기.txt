VAS 2.6.1 — 처음에는 이것만 하세요
====================================

1) 받은 ZIP을 새 폴더에 모두 풉니다.
2) `Run-VAS-System.bat`를 더블클릭합니다.
3) 열린 화면에서 둘 중 하나를 누릅니다.

   새로 만들 때
   `새 프로젝트 만들기` → 내용 작성 → 디자인 선택 → 프로젝트 RAG

   원래 프로그램이 있을 때
   `기존 프로그램 가져오기` → 원래 폴더 선택 → 분석 확인 → 가져오기

다시 이어서 할 때
-----------------
`Run-VAS-System.bat`를 다시 실행하고, 허브의 `내 프로젝트`에서
원하는 프로젝트의 `계속 작업`을 누르세요.

다른 사람에게 줄 때
-------------------
프로젝트 옆 `안전 ZIP`을 누르세요.
원본 코드, 비밀번호, API 키, 개인 사용 기록은 ZIP에 넣지 않습니다.

꼭 알아둘 점
------------
- 원래 프로그램은 자동으로 바꾸거나 지우지 않습니다.
- 새 프로젝트와 가져온 사본은 `workspace\projects` 안에 저장됩니다.
- 기존 프로그램 가져오기에는 Python 3.10 이상이 필요합니다.
- RAG와 사용 기록은 사용자가 켠 뒤에만 이 컴퓨터에서 작동합니다.
- 문제가 생기면 ZIP을 완전히 풀었는지 확인하고 BAT를 다시 실행하세요.

별도 `VAS-Client-Form-*` ZIP은 압축을 푼 뒤 `index.html`을 여세요.
