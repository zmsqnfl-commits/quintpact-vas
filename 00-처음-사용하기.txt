VAS 2.6.2 — 처음에는 이것만 하세요
====================================

1) 받은 ZIP을 새 폴더에 모두 풉니다.
2) Run-VAS-System.bat를 더블클릭합니다.
3) 열린 허브에서 원하는 시작 버튼을 누릅니다.

새로 만들 때
------------
`새 프로젝트 만들기` → 내용 작성 → 작업공간 만들기
만든 프로젝트는 `workspace\projects` 폴더에 저장됩니다.

원래 프로그램을 AI로 이어서 고칠 때
------------------------------------
1. `기존 프로그램 AI로 연결`을 누릅니다.
2. 원래 프로그램 폴더를 선택합니다.
3. `안전하게 분석하기`를 누릅니다.
4. 분석 결과와 보낼 내용을 확인합니다.
5. `AI 전달팩 만들기`를 누릅니다.
6. Codex, Claude, Antigravity 같은 코딩 도구에서 원래 폴더를 엽니다.
7. 저장된 VAS-AI-HANDOFF.json을 첨부하고 복사된 문장을 붙여넣습니다.

원래 프로그램을 VAS 안으로 복사하려면
--------------------------------------
전달 내용 확인 화면 아래 `고급 기능`을 열고 `백업 후 복사·등록`을 누릅니다.
AI 전달팩만 필요하면 복사하지 않아도 됩니다.

꼭 알아둘 점
------------
- 분석 중 원래 프로그램을 실행하거나 바꾸거나 지우지 않습니다.
- 분석 JSON에는 절대 경로, 비밀번호, API 키, 개인화 메모리를 넣지 않습니다.
- `검토 소스 ZIP`은 내가 직접 체크한 텍스트 발췌만 포함합니다.
- 등록된 프로젝트의 활성 RAG는 미리보기로 확인된 항목만 넣습니다.
- Python 3.10 이상이면 Windows판의 JSON·검토 ZIP·복사 기능을 모두 쓸 수 있습니다.
- 웹판은 파일 내용이 없는 분석 JSON만 만듭니다.

다시 이어서 할 때
-----------------
Run-VAS-System.bat를 다시 실행하고 `내 프로젝트`의 `AI 전달팩` 또는 `계속 작업`을 누르세요.

문제가 생기면 ZIP을 완전히 풀었는지 확인하고 BAT를 다시 실행하세요.
별도 VAS-Client-Form-* ZIP은 압축을 푼 뒤 index.html을 여세요.
