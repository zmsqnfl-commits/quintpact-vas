VAS 2.6.3 — 처음에는 이것만 하세요
====================================

1) 받은 ZIP을 새 폴더에 모두 풉니다.
2) Run-VAS-System.bat를 더블클릭합니다.
3) 열린 화면에서 시작 방식을 고릅니다.

새 프로그램을 만들 때
----------------------
`새 프로젝트 만들기` → 질문에 답하기 → 디자인 선택 → JSON 저장

기존 프로그램을 고칠 때
------------------------
`기존 프로그램 AI로 연결` → 원본 폴더 선택 → 안전 분석 → 디자인 선택 → JSON 저장

JSON을 만든 다음
----------------
1. Codex, Claude, Antigravity 같은 코딩 도구를 엽니다.
2. 새 프로젝트는 빈 폴더, 기존 프로그램은 원본 폴더를 작업 폴더로 엽니다.
3. VAS-AI-HANDOFF.json을 첨부합니다.
4. VAS가 복사해 준 프롬프트를 붙여넣습니다.
5. 이제 VAS를 닫아도 됩니다.

다시 설정하려면
---------------
화면 위 `설정`을 누르면 작업 기억을 켜거나 끄고, 잠시 중지하거나 삭제할 수 있습니다.
`사용 방법`을 누르면 이 순서를 언제든 다시 볼 수 있습니다.

꼭 알아둘 점
------------
- 기존 프로그램 분석 중 원본을 실행·수정·삭제하지 않습니다.
- JSON에는 절대 경로, 비밀번호, API 키, 연락처, 작업 기억 원본을 넣지 않습니다.
- 웹판은 파일 내용 없이 이름·크기·구조만 분석합니다.
- Windows 정밀 분석에는 Python 3.10 이상이 필요합니다.
