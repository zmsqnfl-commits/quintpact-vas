# VAS 2.6.4 공통 컨텍스트

## 목적

VAS는 새 프로그램과 기존 프로그램의 설정을 `VAS-AI-HANDOFF.json`으로 정리해 Codex·Claude·Antigravity에 전달하는 로컬 초기 설정 도구입니다.

## 기준

- 단일 작업 원본: `src/`, `docs/`, `scripts/`, `tests/`
- 결과물: 인계 v3 프롬프트·JSON과 결과 v1 JSON의 폐쇄형 반복
- 배포 생성물: `dist/` (Git 제외)
- 파일당 500줄 이하, 외부 CDN·원격 폰트 금지
- 기존 프로젝트 복사·등록 모듈은 호환용으로만 유지

## 흐름

- 새 작업: 시작 → 요구사항 → 디자인 → 승인 RAG → 인계 v3 → 코딩 도구
- 기존 작업: 폴더 위치 → 작업 작성 → 디자인 → 승인 RAG → 인계 v3 → 코딩 도구에서 RBG 분석
- 반복: 결과 v1 → 연결 검증 → 사용자 승인·보완 → 다음 인계
- 작업 기억: 명시적 동의 → 최소 이벤트 → 상단 설정에서 중지·삭제
- 릴리스: Python → 브라우저 → 패키지 검증

## 안전 규칙

- 기존 프로젝트를 실행·수정·삭제하지 않습니다.
- VAS가 기존 프로젝트의 구조·기술 스택을 추정하지 않습니다.
- 폴더 위치는 복사 프롬프트에만 넣고 절대 경로, 비밀값, 연락처, 작업 기억 원본은 JSON에 넣지 않습니다.
- 동의 전 사용 이벤트를 저장하지 않습니다.
- 10회 스트레스 검사는 사용자가 명시한 경우에만 실행합니다.

## 완료 기준

`npm.cmd run test:python`, `npm.cmd run test:browser`, `npm.cmd run test:package`가 통과하고 캐시·비밀·사용자 데이터가 Git에 없어야 합니다.
