# 🧠 Claude Advisor 전략 가이드

이 문서는 Claude API의 **Advisor Strategy** (`advisor-tool-2026-03-01` 베타)를
에이전트 팀에서 활용하기 위한 가이드입니다.

> **적용 조건:** Claude API를 직접 사용하는 환경에서만 실제 적용 가능합니다.
> Antigravity IDE 기본 에이전트에서는 가이드라인 수준으로 참조합니다.

---

## 1. 개념

```
┌─────────────────────────────────────────────────┐
│  실행자 (Executor): Claude Sonnet 4.6            │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  대부분의 기계적 작업을 빠르고 저렴하게 처리       │
│                                                 │
│  ┌─── 복잡한 판단 필요 시 ───┐                    │
│  │                          │                    │
│  ▼                          │                    │
│  ┌────────────────────┐     │                    │
│  │ 자문 (Advisor):     │     │                    │
│  │ Claude Opus 4.6    │─────┘                    │
│  │ 400-700 토큰 가이드 │                           │
│  └────────────────────┘                           │
└─────────────────────────────────────────────────┘
```

- **실행자 (Executor):** 빠르고 저비용인 모델이 일상 작업을 처리
- **자문 (Advisor):** 복잡한 판단 시에만 고성능 모델이 개입하여 가이드 제공
- **비용 효율:** Opus 비용은 자문 토큰(400~700)에만 발생

## 2. 에이전트별 Advisor 활용 시나리오

| 에이전트 | Advisor 호출 시점 | 예시 질문 |
|----------|------------------|----------|
| **Architect** | 아키텍처 선택 갈림길 | "마이크로서비스 vs 모놀리스, 이 프로젝트에 적합한 구조는?" |
| **Designer** | 디자인 시스템 방향 결정 | "이 프로젝트에 Atomic Design vs BEM, 어떤 접근이 적합한가?" |
| **Implementer** | 복잡한 알고리즘/패턴 결정 | "Observer vs Pub/Sub 중 여기서 적절한 패턴은?" |
| **Reviewer** | 잠재적 성능·보안 이슈 판단 | "이 메모리 패턴이 프로덕션에서 누수를 일으킬까?" |
| **Tester** | 엣지 케이스 발견 | "이 함수에서 놓치고 있는 중요한 테스트 시나리오는?" |
| **Security** | 취약점 심각도 판정 | "이 입력 검증 누락이 실제 공격 벡터가 될 수 있는가?" |

## 3. 사용 규칙

### 호출해야 하는 경우
- 아키텍처 설계 시 2개 이상 선택지가 있을 때
- 성능 최적화 방향을 결정해야 할 때
- 보안 취약점의 심각도를 판단해야 할 때
- 기존 코드의 리팩토링 전략을 수립할 때

### 호출하면 안 되는 경우
- 일상적 코드 작성 및 파일 수정
- 단순 문법 오류 수정
- 변수명 변경, 주석 추가 등 기계적 작업
- 이미 결정된 설계에 따른 구현

### 제한 사항
- 한 세션(워크플로우 사이클)당 **최대 3회**
- 호출 전 질문 의도를 **1~2문장으로** 명확히 정리
- 호출 결과는 인계 포맷에 `[Advisor 참조]` 태그로 포함

## 4. API 설정 참조

```json
{
  "model": "claude-sonnet-4-6",
  "tools": [
    {
      "type": "advisor_20260301",
      "name": "advisor",
      "model": "claude-opus-4-6"
    }
  ],
  "max_uses": 3,
  "anthropic-beta": "advisor-tool-2026-03-01"
}
```

## 5. 인계 시 Advisor 사용 기록

Advisor를 호출한 경우, 인계 포맷에 아래를 추가합니다:

```
[인계] [현재 에이전트] → [다음 에이전트]
- 완료: [한 줄 요약]
- 변경/생성 파일: [파일 목록]
- [Advisor 참조] 질문: [질문 요약] / 답변 요약: [핵심 가이드]
- 특이사항: [주의사항 / 없으면 '없음']
```
