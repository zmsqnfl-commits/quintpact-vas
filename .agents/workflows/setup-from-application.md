# /setup-from-application

1. 신청서 JSON의 스키마와 필수값을 검증합니다.
2. 프로젝트 이름을 안전한 폴더명으로 정규화합니다.
3. `workspace/projects/<name>/`에 새 프로젝트를 만듭니다.
4. 요구사항, 선택한 테마 상태, 작업 체크리스트를 프로젝트 내부 문서로 저장합니다.
5. 프로젝트 레지스트리와 선택적 지식 색인을 갱신합니다.
6. 생성 결과와 다음 작업 링크를 허브에 표시합니다.

신청서 첨부 파일은 이름·크기·형식 메타데이터만 다루며 파일 내용은 자동 업로드하지 않습니다.
