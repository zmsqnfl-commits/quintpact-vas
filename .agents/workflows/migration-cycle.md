# /migration-cycle

기존 프로그램을 VAS에 안전하게 연결하는 흐름입니다.

1. 사용자가 VAS에서 원래 프로젝트 폴더를 선택합니다.
2. VAS가 파일을 실행하거나 변경하지 않고 구조·스택·위험 요소를 분석합니다.
3. 사용자가 작업 내용과 디자인을 확인합니다.
4. VAS가 `VAS-AI-HANDOFF.json`과 붙여넣을 프롬프트를 만듭니다.
5. 사용자가 Codex·Claude·Antigravity에서 원래 폴더와 JSON을 함께 엽니다.
6. 코딩 도구가 실제 파일을 다시 확인한 뒤 안전한 변경 계획을 세웁니다.

기존 복사·등록·롤백 모듈은 호환용 내부 기능으로만 남깁니다. VAS 기본 화면은 원본을 실행·수정·삭제하지 않습니다.
