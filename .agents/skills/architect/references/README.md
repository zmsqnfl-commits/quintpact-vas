# 아키텍처 패턴 레퍼런스

이 디렉터리는 Architect 에이전트가 설계 시 참조하는 아키텍처 패턴 문서를 저장합니다.
`/setup-from-application` 실행 시 프로젝트 유형에 맞는 패턴 문서가 자동 생성됩니다.

## 일반 참조 패턴

| 패턴 | 적용 시나리오 |
|------|-------------|
| MVC / MVVM | 웹 애플리케이션, GUI 앱 |
| 파이프라인 (Pipeline) | 데이터 처리, 영상/음성 분석 |
| 이벤트 기반 (Event-Driven) | 실시간 시스템, IoT |
| 마이크로서비스 | 대규모 분산 시스템 |
| 레이어드 (Layered) | 일반적인 비즈니스 로직 |

> 프로젝트별 구체적 패턴은 `/setup-from-application` 시 여기에 추가됩니다.
