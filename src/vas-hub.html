프로젝트 시작

새 프로젝트 만들기요구사항을 정리하고 새 작업 공간을 준비합니다. 기존 프로그램 가져오기원본은 그대로 두고 안전한 복사본으로 시작합니다.

처음 사용하는 순서

  1. 1시작 방식 선택새 프로젝트 또는 기존 프로그램을 선택합니다.
  2. 2디자인과 작업 설정원하는 모습과 다음 작업을 정합니다.
  3. 3추천과 기록으로 이어서 작업필요한 문서와 최근 흐름을 다시 찾습니다.

작업 도구

디자인 스튜디오화면의 색상과 구성을 정합니다. 내부 지식 검색VAS 문서와 프로젝트 내용을 찾습니다. 내 사용 기록최근 작업과 저장된 기록을 확인합니다.

내 프로젝트
