# src/ — 소스 코드 폴더

이 폴더는 에이전트들이 코드와 에셋을 작성하는 **작업 공간**입니다.

## 규칙
- **Implementer**가 `src/` 전체에서 코드를 생성/수정합니다.
- **Designer**는 `src/assets/`에서만 디자인 에셋(토큰, 아이콘, 이미지)을 생성합니다.
- 파일당 **500줄 이하**를 유지합니다.
- 검증이 완료되면 Security 에이전트가 `final/`로 복사합니다.
- 이 폴더의 코드는 항상 **최신 작업본**입니다.

## 구조
`/setup-from-application` 실행 시 프로젝트에 맞는 하위 디렉터리가 생성됩니다.

```
src/
├── assets/     ← 디자인 에셋 (Designer 담당: 토큰, 아이콘, 이미지)
├── core/       ← 핵심 비즈니스 로직
├── ui/         ← 사용자 인터페이스
├── utils/      ← 유틸리티 함수
└── config/     ← 설정 파일
```

> 위 구조는 예시이며, Architect 에이전트의 설계에 따라 달라집니다.
