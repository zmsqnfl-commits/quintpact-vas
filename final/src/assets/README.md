# 🎨 src/assets/ — 디자인 에셋 폴더

이 폴더는 **Designer 에이전트가 생성하는 디자인 에셋**을 저장하는 공간입니다.

## 저장 대상
- `design-tokens.json` 또는 CSS 변수 파일 — 색상, 폰트, 간격 등 디자인 토큰
- 아이콘 파일 (SVG, PNG 등)
- 이미지 에셋 (WebP, PNG 등)
- 기타 시각 리소스

## 접근 권한
- **Designer:** ✅ 쓰기 (주 담당)
- **Implementer:** ✅ 읽기 + 코드에서 참조
- **기타 에이전트:** ✅ 읽기만

## 규칙
- 이 폴더의 파일은 Designer가 생성하고, Implementer가 코드에서 참조합니다.
- 불필요한 에셋은 정리하여 프로젝트 크기를 최소화합니다.
- `final/` 배포 시 Security가 이 폴더의 에셋도 함께 복사합니다.
