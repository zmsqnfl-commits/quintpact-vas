# 기존 프로그램 AI로 연결

1. 원본 프로그램 폴더를 선택합니다.
2. AI에게 맡길 일을 적습니다.
3. 읽기 전용 분석 결과와 제외 항목을 확인합니다.
4. 디자인 방향과 코딩 도구를 선택합니다.
5. `VAS-AI-HANDOFF.json`을 저장하고 프롬프트를 복사합니다.
6. 코딩 도구에서 원본 폴더를 열고 JSON을 첨부합니다.

분석 중 원본을 실행·수정·삭제하지 않습니다. Git 내부, 캐시, 의존성, 링크, 바이너리, 비밀값 후보, 절대 경로는 제외합니다. 웹판은 파일 내용 없이 이름·크기·구조만 읽습니다.

기존 복사·등록·롤백 모듈은 이전 프로젝트 호환을 위해 남겨두지만 일반 화면에서는 사용하지 않습니다.
