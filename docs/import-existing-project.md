# 기존 프로그램 AI로 연결

1. **폴더 위치**를 눌러 원본 폴더 위치를 고릅니다.
2. 프로젝트 이름과 AI에게 맡길 일을 적습니다.
3. 디자인 방향과 코딩 도구를 선택합니다.
4. 완성된 프롬프트를 복사해 Codex·Claude·Antigravity에 붙여넣습니다.
5. 설정 보관이 필요하면 `VAS-AI-HANDOFF.json`도 프로젝트 최상위에 저장합니다.
6. AI 작업 뒤 `VAS-AI-RESULT.json`을 다시 불러오면 남은 작업을 다음 인계로 연결합니다.

VAS는 선택한 위치만 프롬프트에 넣고 폴더 내부는 읽지 않습니다. 절대 경로는 JSON·작업 기억에 저장하지 않습니다. 실제 구조 분석과 수정은 코딩 AI가 RBG 규칙으로 담당합니다.

기존 복사·등록·롤백 모듈은 이전 프로젝트 호환을 위해 남겨두지만 일반 화면에서는 사용하지 않습니다.
