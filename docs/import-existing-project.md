# 기존 프로그램을 AI로 연결하기

## 가장 쉬운 방법

1. `Run-VAS-System.bat`를 실행합니다.
2. 허브에서 **기존 프로그램 AI로 연결**을 누릅니다.
3. 원래 프로그램 폴더를 선택하고 **안전하게 분석하기**를 누릅니다.
4. 기술 스택, 파일 수, 제외 항목을 확인합니다.
5. **분석 JSON** 또는 직접 고른 파일만 담는 **검토 소스 ZIP**을 선택합니다.
6. 사용할 코딩 도구를 고르고 **AI 전달팩 만들기**를 누릅니다.
7. 코딩 도구에서 원본 폴더를 연 뒤 `VAS-AI-HANDOFF.json`을 첨부하고 복사된 프롬프트를 붙여넣습니다.

VAS가 코딩 도구 계정이나 API 키를 요구하거나 자동 전송하지 않습니다.

## 무엇이 들어가나요?

- 상대 경로 파일 목록, 기술 스택, 진입점, 안전하게 읽은 매니페스트 정보
- 사용자가 적은 작업 요청
- 등록 프로젝트에서 활성화된 요구사항·디자인·프로젝트 RAG의 미리보기 항목
- 원본 폴더를 실제 수정 기준으로 사용하라는 공통 안내

절대 경로, 개인화 메모리, `.env`, 키·토큰, Git 내부, 의존성, 캐시, 링크, 바이너리는 제외합니다. 검토 소스 ZIP은 사용자가 체크한 안전한 텍스트의 제한된 발췌만 담습니다.

## 웹판과 Windows판

- 웹판: 파일 이름과 크기로 내용 없는 분석 JSON을 만듭니다.
- Windows판: 매니페스트 분석, 등록 프로젝트 RAG, JSON 재검증, 검토 소스 ZIP을 제공합니다.
- 원본 소스가 분석 후 바뀌면 저장을 중단하고 다시 분석하도록 안내합니다.

## VAS 안으로 복사하려면

전달 내용 확인 화면 아래 **고급 기능**을 엽니다. 확인 후 백업 → staging 복사 → SHA-256 검증 → `workspace/projects/` 등록 순서로 진행합니다. 기본 AI 전달 흐름에는 복사가 필요하지 않습니다.

가져오기 결과가 맞지 않으면 완료 화면의 롤백을 사용합니다. 원본에는 영향을 주지 않습니다.
