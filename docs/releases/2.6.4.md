# VAS 2.6.4

VAS를 `설정 → 프롬프트 복사 → 코딩 AI 작업`에 집중한 초기 설정 도구로 정리했습니다.

- 새 프로젝트·기존 프로그램 모두 프롬프트 복사를 기본 동작으로 통일하고 JSON 저장은 선택 기능으로 유지
- 기본 프롬프트에서 결과 회수·반복 지시·RAG 원문·내부 인계 ID를 제거
- 코딩 AI가 원본 파일을 먼저 확인하는 RBG 지침 명시
- 결과 불러오기·RAG 검토·반복 지시는 기본 UI와 프롬프트에서 제거하고 호환 모듈로만 유지
- 호환 결과를 직접 불러올 때는 ID·SHA-256·sourceType·안전 플래그를 엄격히 검사
- 디자인 새 창 설정 동기화, 비저장 폴더 위치 선택, 모바일 도움말과 키보드 포커스 보강
- 프리셋과 무관하게 VAS 설정 화면의 Awwwards Editorial 대비 유지
- 외부 요청 없이 오프라인·저장소 차단·모바일 흐름 검증
