# VAS 2.6.4 개발 규칙

- HTML/CSS/Vanilla JS, Python, PowerShell만 사용합니다.
- 파일당 500줄 이하, 외부 CDN·원격 폰트 금지입니다.
- 코어 원본은 `src/`, `docs/`, `scripts/`, `tests/`입니다.
- 신규·기존 흐름은 `VAS-AI-HANDOFF.json` v3으로 작업을 보내고 `VAS-AI-RESULT.json` v1로 결과를 받습니다.
- 결과는 인계 ID·반복 번호·해시를 확인한 뒤에만 다음 인계로 연결합니다.
- RAG는 사용자가 승인한 안전 요약 최대 3개만 포함하고, 코딩 AI는 RBG로 실제 파일을 먼저 읽습니다.
- 기존 프로그램의 구조·기술 스택은 VAS에서 추정하지 않고 코딩 AI가 원본 폴더에서 확인합니다.
- 기존 프로그램의 폴더 위치는 복사 프롬프트에만 포함하고 JSON·작업 기억에는 저장하지 않습니다.
- 개인화는 명시적 동의가 있어야 하며 최종 JSON에 원본 기록을 넣지 않습니다.
- 기존 복사·등록 모듈은 호환용으로 보존하되 기본 UI에 노출하지 않습니다.
- 평소에는 Python·브라우저·패키지 테스트를 실행합니다.
- 10회 스트레스 검사는 명시적 요청 때만 실행합니다.
